Introduction

본 API는 차량 상태 제어 기능을 제공한다.

Type Definitions

DoorLockMode

도어락(door lock) 모드.

enum DoorLockMode { "UNLOCK", "LOCK" };
  • UNLOCK: 차량문 Unlock.
  • LOCK: 차량문 Lock.

TrunkStatus

트렁크 상태.

enum TrunkStatus { "OPENED", "CLOSED" };
  • OPENED: Opened 상태.
  • CLOSED: Closed 상태.

SeatbeltPosition

안전벨트 위치.

enum SeatbeltPosition { "DRIVER", "PASSENGER" };
  • DRIVER: 운전석 안전벨트.
  • PASSENGER: 조수석 안전벨트.

SeatbeltStatus

안전벨트 상태.

enum SeatbeltStatus { "UNBELTED", "BELTED" };
  • UNBELTED: Unbelted 상태.
  • BELTED: Belted 상태.

LampType

차량 램프 종류.

enum LampType { "TAIL", "HEAD_LOW", "HEAD_HIGH", "FOG", "HAZARD" };
  • TAIL: 미등.
  • HEAD_LOW: 하향등.
  • HEAD_HIGH: 상향등.
  • FOG: 안개등.
  • HAZARD: 비상등.

LampStatus

차량 램프 상태.

enum LampStatus { "OFF", "ON" };
  • OFF: 차량 램프 Off.
  • ON: 차량 램프 On.

AirbagStatus

에어백 상태.

enum AirbagStatus { "OFF", "ON" };
  • OFF: 에어백 Off.
  • ON: 에어백 On.

DoorPosition

차량 문 위치.

enum DoorPosition { "FRONT_LEFT", "FRONT_RIGHT", "REAR_LEFT", "REAR_RIGHT" };
  • DOOR_FRONT_LEFT: 앞좌측 차량 문.
  • DOOR_FRONT_RIGHT: 앞우측 차량 문.
  • DOOR_REAR_LEFT: 뒤좌측 차량 문.
  • DOOR_REAR_RIGHT: 뒤우측 차량 문.

DoorStatus

차량 문 상태.

enum DoorStatus { "UNLOCKED", "LOCKED", "OPENED", "CLOSED" };
  • UNLOCKED: Unlocked 상태.
  • LOCKED: Locked 상태.
  • OPENED: Opened 상태.
  • CLOSED: Closed 상태.

PowerType

전원 종류.

enum PowerType { "ENGINE", "ACC", "IGNITION" };
  • ENGINE: 엔진.
  • ACC: ACC.
  • IGNITION : 점화 장치.

PowerStatus

전원 상태.

enum PowerStatus { "OFF", "ON" };
  • OFF: 전원 Off.
  • ON: 전원 On.

DiagnosisType

진단 장치 종류.

enum DiagnosisType { "LOW_FUEL", "EMS" };

  • LOW_FUEL: 연료 부족 경고등.
  • EMS: 엔진 경고등(EMS: Engine Management System).

DiagnosisStatus

진단 장치 상태.

enum DiagnosisStatus { "OFF", "ON" };
  • OFF: 진단 장치 Off.
  • ON: 진단 장치 On.

Features

When the feature
  • http://hyundai.com/api/status
or any of the features hierarchically under that feature is successfully requested, the interface Status is instantiated, and the resulting object appears in the global namespace as motor.status.
This is the list of URIs used to declare this API's features, for use in the widget config.xml. For each URL, the list of functions covered is provided.

setDoor(), watchDiagnosis(), clearWatchDiagnosis(), watchGPSData(), clearWatchGPSData()
