Introduction

본 API는 차량 정보를 제공한다.

Type Definitions

EngineType

차량 엔진 종류.

enum EngineType { "GASOLINE", "DIESEL", "LPG", "HYBRID", "ELECTRIC" };
  • GASOLINE: 가솔린 엔진.
  • DIESEL: 디젤 엔진.
  • LPG: LPG 엔진.
  • HYBRID: 하이브리드 엔진.
  • ELECTRIC: 전기 엔진.

GearType

차량 변속기 종류.

enum GearType { "AT", "CVT", "DCT", "MT" };
  • AT: Automatic Transmission.
  • CVT: Continuously Variable Transmission.
  • DCT: Double or Dual Clutch Transmission.
  • MT: Manual Transmission.

HeadUnitGrade

차량 Head unit 등급.

enum HeadUnitGrade { "STANDARD", "PREMIER" };
  • STANDARD: 표준형.
  • PREMIER : 고급형.

VehicleRegion

차량 지역향 코드.

enum VehicleRegion { "KR", "US", "CN", "EU" };
  • KR: 한국향(KOREA).
  • US: 북미향(NORTH AMERICA).
  • CN: 중국향(CHINA).
  • EU: 유럽향(EUROPE).

VehicleManufacturer

차량 제조사.

enum VehicleManufacturer { "HYUNDAI", "KIA" };
  • HYUNDAI : 현대.
  • KIA: 기아.

Interfaces

InfoObject

Defines what is instantiated in the motor object.
There will be a motor.info object that allows accessing the functionality of the info module.
readonly attribute Info info

Features

When the feature
  • http://hyundai.com/api/info
or any of the features hierarchically under that feature is successfully requested, the interface Info is instantiated, and the resulting object appears in the global namespace as motor.info.
This is the list of URIs used to declare this API's features, for use in the widget config.xml. For each URL, the list of functions covered is provided.

http://hyundai.com/api/info.get.pubilc
Access to all the feature except the following methods: getVIN()

http://hyundai.com/api/info.get.pivate
Access to all the feature except the following methods:
getEngineType(), getGearType(), getCenterRegisterStatus(), getVehicleModelName(), getHeadUnitGrade(), getHeadUnitSerialNumber(), getVehicleRegion(), getVehicleManufacturer(), getAPIVersion()

http://hyundai.com/api/info
Access to all the features in this module.
