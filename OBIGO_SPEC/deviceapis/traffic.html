Introduction

This API provides the TMC(Traffic Message Channel) information.

Type Definitions

TMCType

The type of the TMC device.

enum TMCType { "DAB", "RDS", "SIRIUS", "TPEG", "XM" };
  • DAB: Digital Audio Broadcasting.
  • RDS: Radio Data System.
  • SIRIUS: Sirius Satellite Radio.
  • TPEG: Transport Protocol Expert Group.
  • XM: XM Satellite Radio.

TPEGDataFileStatus

The status of the TPEG data file.

enum TPEGDataFileStatus { "CREATED", "WILL_BE_REMOVED", "REMOVED" };
  • CREATED: The TPEG data file was created.
  • WILL_BE_REMOVED: The TPEG data file will be removed.
  • REMOVED: The TPEG data file was removed.

TPEGDataFile

Define the attributes of a TPEG data file.

The status of a TPEG data file.

The pathname of a TPEG data file.
