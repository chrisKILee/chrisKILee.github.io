Introduction

This API provides the status information of a vehicle.

Type Definitions

DoorLockMode

The mode of a door lock.

enum DoorLockMode { "UNLOCK", "LOCK" };
  • UNLOCK: A door lock is unlocked.
  • LOCK: A door lock is locked.

TrunkStatus

The status of a trunk.

enum TrunkStatus { "OPENED", "CLOSED" };
  • OPENED: A trunk was opened.
  • CLOSED: A trunk was closed.

SeatbeltPosition

The position of a seat belt.

enum SeatbeltPosition { "DRIVER", "PASSENGER" };
  • DRIVER: A driver's seat.
  • PASSENGER: A passenger's seat.

SeatbeltStatus

The status of a seat belt.

enum SeatbeltStatus { "UNBELTED", "BELTED" };
  • UNBELTED: A seat belt was unbelted.
  • BELTED: A seat belt was belted.

LampType

The type of a vehicle lamp.

enum LampType { "TAIL", "HEAD_LOW", "HEAD_HIGH", "FOG", "HAZARD" };
  • TAIL: Tail lights.
  • HEAD_LOW: Low beam headlights.
  • HEAD_HIGH: High beam headlights.
  • FOG: Fog lights.
  • HAZARD: hazard lights.

LampStatus

The status of a vehicle lamp.

enum LampStatus { "OFF", "ON" };
  • OFF: A vehicle lamp was turned off.
  • ON: A vehicle lamp was turned on.

AirbagStatus

The status of an airbag.

enum AirbagStatus { "OFF", "ON" };
  • OFF: An airbag was turned off.
  • ON: An airbag was turned on.

DoorPosition

The position of a vehicle door.

enum DoorPosition { "FRONT_LEFT", "FRONT_RIGHT", "REAR_LEFT", "REAR_RIGHT" };
  • DOOR_FRONT_LEFT: The front left door of a vehicle.
  • DOOR_FRONT_RIGHT: The front right door of a vehicle.
  • DOOR_REAR_LEFT: The rear left door of a vehicle.
  • DOOR_REAR_RIGHT: The rear right door of a vehicle.

DoorStatus

The status of a vehicle door.

enum DoorStatus { "UNLOCKED", "LOCKED", "OPENED", "CLOSED" };
  • UNLOCKED: A vehicle door was unlocked.
  • LOCKED: A vehicle door was locked.
  • OPENED: A vehicle door was opened.
  • CLOSED: A vehicle door was closed.

PowerType

The type of a vehicle power.

enum PowerType { "ENGINE", "ACC", "IGNITION" };
  • ENGINE: The engine.
  • ACC: The ACC.
  • IGNITION: The ignition.

PowerStatus

The status of a vehicle power.

enum PowerStatus { "OFF", "ON" };
  • OFF: A vehicle power was turned off.
  • ON: A vehicle power was turned on.

DiagnosisType

The type of a vehicle diagnostic device.

enum DiagnosisType { "LOW_FUEL", "EMS" };

  • LOW_FUEL: The fuel management system.
  • EMS: The engine management system.

DiagnosisStatus

The status of a vehicle diagnostic device.

enum DiagnosisStatus { "OFF", "ON" };
  • OFF: A vehicle diagnostic device was turned off.
  • ON: A vehicle diagnostic device was turned on.

EngineOilStatus

The status of an engine oil.

enum DoorStatus { "OVER_MAX","MAX", "OK", "MIN", "UNDER_MIN" };
  • OVER_MAX: An engine oil at over maximun level.
  • MAX: An engine oil at maximun level.
  • OK: An engine oil at OK level.
  • MIN: An engine oil at minimum level.
  • UNDER_MIN: An engine oil at under minimum level.

TirePosition

The position of a tire.

enum TirePosition { "FRONT_LEFT", "FRONT_RIGHT", "REAR_LEFT", "REAR_RIGHT" };
  • DOOR_FRONT_LEFT: The front left tire of a vehicle.
  • DOOR_FRONT_RIGHT: The front right tire of a vehicle.
  • DOOR_REAR_LEFT: The rear left tire of a vehicle.
  • DOOR_REAR_RIGHT: The rear right tire of a vehicle.

VehicleWindowPosition

The position of a vehicle window.

enum TirePosition { "DRIVER", "FRONT_PASSENGER", "BEHIND_DRIVER", "BEHIND_PASSENGER" };
  • DRIVER: The driver window of a vehicle.
  • FRONT_PASSENGER: The front passenger window of a vehicle.
  • BEHIND_DRIVER: The window behind the driver of a vehicle.
  • BEHIND_PASSENGER: The window behind front passenger of a vehicle.
