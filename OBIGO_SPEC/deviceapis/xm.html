Introduction

This API provides functionalities to control the XM device.

Type Definitions

AutoTuningOption

The option of auto tunning.

enum AutoTuningOption { "UP", "DOWN" };
  • UP: Tunes up XM frequency.
  • DOWN: Tunes down XM frequency.

Interfaces

XMObject

There will be a motor.xm object that allows accessing the functionality of the XM module.
readonly attribute XM xm

Turns the XM radio device on or off.
Signature

Return value

Exceptions

with error code TYPE_MISMATCH_ERR if the input parameter is not compatible with the expected type for that parameter.
with error code NOT_SUPPORTED_ERR if this function is not supported.
boolean state
True to power on the XM radio device, false to power it off.
optional SuccessCallback? successCallback
The method to be invoked when the asynchronous call completes successfully.
optional ErrorCallback? errorCallback
The method to be invoked when an error occurs.
						
						motor.xm.setPowered(true, function() {
							console.log("The XM radio device powered on success.");
						},
						function(e) {
							console.log("Failed to power on the XM radio device: " + e.message);
						});
