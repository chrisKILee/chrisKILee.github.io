Introduction

This API provides the vehicle information.

Type Definitions

EngineType

The type of a vehicle engine.

enum EngineType { "GASOLINE", "DIESEL", "LPG", "HYBRID", "ELECTRIC" };
  • GASOLINE: The gasoline engine.
  • DIESEL:The diesel engine.
  • LPG: The LPG engine.
  • HYBRID: The hybrid engine.
  • ELECTRIC: The electric engine.

GearType

The type of a vehicle transmission.

enum GearType { "AT", "CVT", "DCT", "MT" };
  • AT: The automatic transmission.
  • CVT: The continuously variable transmission.
  • DCT: The double or dual clutch transmission.
  • MT: The manual transmission.

HeadUnitGrade

The grade of a vehicle's head unit.

enum HeadUnitGrade { "LOW", "HIGH" };
  • LOW: The low grade.
  • HIGH: The high grade.

VehicleRegion

The region code of a vehicle.

enum VehicleRegion { "KR", "US", "CN", "EU" };
  • KR: Korea.
  • US: North America.
  • CN: China.
  • EU: Europe.

VehicleManufacturer

The manufacturer of a vehicle.

enum VehicleManufacturer { "XXX", "YYY" };
  • XXX: XXX motor company.
  • YYY: YYY motor company.

Features

When the feature
  • http://obigo.com/api/info
or any of the features hierarchically under that feature is successfully requested, the interface Info is instantiated, and the resulting object appears in the global namespace as motor.info.
This is the list of URIs used to declare this API's features, for use in the widget config.xml. For each URL, the list of functions covered is provided.

http://obigo.com/api/info.get.pubilc
Access to all the feature except the following methods: getVIN()

http://obigo.com/api/info.get.pivate
Access to all the feature except the following methods:
getEngineType(), getGearType(), getCenterRegisterStatus(), getVehicleModelName(), getHeadUnitGrade(), getHeadUnitSerialNumber(), getVehicleRegion(), getVehicleManufacturer(), getAPIVersion()

http://obigo.com/api/info
Access to all the features in this module.
