### short-cut
- 편집기 분할
  - `⌃` + `⌘` +  `방향키`
  - `⌘` + `숫자키`
  - `⌘` + `\`
- multi-selection
  - `⇧` + `⌥`
- 줄 이동
  - `⌥` + `방향키`
- 줄 복사
  - `⇧` + `⌥` + `방향키`
- tab 이동
  - `⌘` + `⌥` + `방향키`
- multi Selection
  - `⇧` + `⌘` + `⌥` + `방향키`
  - `⇧` + `⌘` +`마우스`

### setting Sync
- markget place : Settings Sync
- Upload Key :  `⇧` + `⌥` + `U`
- Download Key : `⇧` + `⌥` + `D`
